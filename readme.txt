Adds your Google Universal Analytics tracking code to your WordPress site.

== Description ==

LP Google Analytics allows you to easily add your Google Universal Analytics tracking code on all your pages. You just specify your ID, choose whether to place the code in header or footer and you are done.
Please note that this plugin inserts tracking code only! View your stats in your Google account.

== Installation ==

1. Download `lp-google-analytics.zip`
2. Upload the contents of `lp-google-analytics.zip` to the `/wp-content/plugins/` directory.
3. Activate the plugin through the `Plugins` menu in WordPress
